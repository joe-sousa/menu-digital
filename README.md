This Android application, developed with Kotlin, serves as a digital menu system aimed at simplifying the process of adding and removing dishes/beverages, calculating prices, and closing orders. 

## Features

- **Easy Navigation:** The app offers a user-friendly interface, allowing customers to navigate through various categories effortlessly.
- **Dynamic Menu Management:** Add or remove items dynamically, enabling restaurant staff to update the menu instantly.
- **Price Calculation:** The application automates price calculation based on the items selected, ensuring accurate billing.
- **Order Management:** Facilitates the process of closing orders, providing a seamless experience for both customers and staff.

## Getting Started

1. **Clone the Repository:**
   ```
   git clone https://github.com/joe-sousa/menu-digital.git
   ```

2. **Open with Android Studio:**
   Open Android Studio and select "Open an existing Android Studio project." Navigate to the directory where you cloned the repository and select it.

3. **Run the App:**
   Connect your Android device or use an emulator and click on the "Run" button in Android Studio to install and launch the app.

## Usage

1. **Browse the Menu:**
   Explore the menu categories and select items by tapping on them.

2. **Add or Remove Items:**
   Easily add or remove items from your order with a simple tap.

3. **Calculate Prices:**
   The app dynamically calculates the total price of your order as you add or remove items.

4. **Close Order:**
   When ready, proceed to close your order, and the app will finalize the billing process.
